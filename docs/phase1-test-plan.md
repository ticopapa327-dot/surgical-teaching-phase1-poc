## 测试用例

| 编号 | 测试项 | 操作 | 通过标准 | 结果 |
|---|---|---|---|---|
| T1 | 设备枚举 | 点击授权并刷新设备。 | 可识别目标 USB 视频输入和音频输入。 |  |
| T2 | 四路预览 | 为 4 个通道选择设备或模拟源并启动全部预览。 | 4 路画面均可稳定显示。 |  |
| T3 | 单路录制 | 仅选择通道 1 录制 5 分钟。 | 停止后生成录像，应用内可回放。 |  |
| T4 | 多路录制 | 同时选择 4 路录制 30 分钟。 | 录像文件完整，预览无明显卡死。 |  |
| T5 | 音频录制 | 选择 USB 麦克风并录制。 | 回放能听到音频，无明显持续回声。 |  |
| T6 | 文件索引 | 完成多次录制后刷新索引。 | 索引记录与文件一致，可定位文件。 |  |
| T7 | 异常处理 | 录制中停止单个通道和全部通道。 | 文件可正常关闭并回放。 |  |
| T8 | 录像删除 | 完成一次录制后在录像索引中点击删除。 | 录像记录从索引中移除，列表回到空状态。 |  |
| T9 | 录像导出 | 完成一次录制后在录像索引中点击导出。 | 可导出或下载对应录像文件。 |  |
| T10 | UVC 云台/镜头控制 | 将通道切换为 USB/摄像头设备并启动预览，点击方向和变倍控制。 | 支持 UVC PTZ 的设备可响应控制；不支持时软件给出明确提示。 |  |
